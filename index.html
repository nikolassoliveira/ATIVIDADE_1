ExaGear: emulador de Windows para Android é capaz de rodar GTA IV e Skyrim

O que é o Exagear?

Saiba mais....

O Exagear é uma camada de emulação que traduz instruções x86 para ARM, o que permite que entusiastas de programação usem suas bases para criar novos projetos.

Desenvolvido pelo grupo eltechs o emulador recebeu esta semana uma nova atualização informal, criada por um membro da comunidade. Chamado de ExaGear V2 , o mod agora permite que o emulador rode jogos como GTA IV, The Elder Scrolls V: Skyrim, Far Cry 2, necessitando apenas de um celular com Android.
